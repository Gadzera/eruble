# Платформа цифрового рубля B2B — ООО «НПК Технопром-Сервис»

Операционная оболочка для работы с цифровым рублём через платформу Банка России. Разработана для корпоративных клиентов, осуществляющих B2B-расчёты в цифровой валюте центрального банка (ЦВЦБ).

## Функциональность

- Управление кошельком ЦР: остаток, история операций, пополнение и вывод
- B2B-переводы: создание, согласование (dual-control), отправка в платформу ЦБ
- Реестры платежей: массовые выплаты (зарплата, поставщики, налоги) через CSV/XLSX/XML
- Операции QR: приём входящих клиентских платежей
- Управление контрагентами со справочником ЦР-счетов
- Audit trail: полное неизменяемое журналирование всех событий
- RegTech: выгрузка отчётности по форматам ЦБ
- REST API с OpenAPI 3.1 спецификацией и Postman-коллекцией
- Мобильная адаптация, тёмная/светлая тема

## Стек технологий

- **Frontend**: Next.js 15 (App Router), React, TypeScript, Tailwind CSS, shadcn/ui, Recharts
- **Backend**: Next.js Route Handlers, Drizzle ORM, SQLite (better-sqlite3)
- **Auth**: сессионная авторизация на edge middleware

## Запуск (разработка)

```bash
npm install
npm run db:seed   # инициализация демо-данных
npm run dev       # сервер на http://localhost:3000
```
